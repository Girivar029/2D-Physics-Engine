# 2D Game Engine Built In C++

## Project Specifications:
This project is a very *low consumption, highly customizable*  engine which can be customized for any game. The main goal of this engine is to reduce game load and improve performance by removing any *unnecesary features*.

## Project Features:
- *Working Audio Features*: Basic Audio Controls With Multiple Channels.
- *Input Handling*: Flexible And Smooth Input Handling.
- *Graphics Rendering*: Basic graphics rendering for 2d games.
- *Asset Management System*: Controls the arrangement and upkeep of files.
- *Typoid System*: No built it types of game objects rather the user can make custom game objects for their specific needs.
- *Customization*: The user is able to customize the engine to any level and get results specific to them and can make more optimized large scale games.

## Tech Stack Used:
- C++ for writing.
- SLD2 for basic graphics rendering.
- SLD2 for Audio.
- CMake for compilation and project management.
